
ステップ１：シミュレーションプロジェクトの作成
==============================================

ここでは、ドローンのシミュレーションを始めるための準備を行います。基本的な操作は `Tankチュートリアル <https://choreonoid.org/ja/manuals/latest/simulation/tank-tutorial/index.html>`_ と同様です。以下に参考ページのリンクを示します。

ドローンモデルの作成
--------------------

まず、ドローンモデルを作成します。テキストエディタを起動して、以下を記述してください。
`ドローンモデルの記述 <https://raw.githubusercontent.com/k38-suzuki/hairo-world-plugin/main/sample/tutorial/Drone/drone.body>`_ を参照してください。

ドローンモデルは、ファイル名を"drone.body"として保存してください。

Choreonoidの起動
----------------

`Choreonoidの起動 <https://choreonoid.org/ja/manuals/latest/simulation/tank-tutorial/step1.html#choreonoid>`_ を参照してください。

ワールドアイテムの作成
----------------------

`ワールドアイテムの作成 <https://choreonoid.org/ja/manuals/latest/simulation/tank-tutorial/step1.html#id3>`_ を参照してください。

ドローンモデルの読み込み
------------------------

ここでドローンモデル"drone.body"を読み込みます。ボディモデルの読み込みについては

`モデルの読み込み <https://choreonoid.org/ja/manuals/latest/simulation/tank-tutorial/step1.html#tank>`_ を参照してください。

床モデルの読み込み
------------------

`床モデルの読み込み <https://choreonoid.org/ja/manuals/latest/simulation/tank-tutorial/step1.html#id4>`_ を参照してください。

シミュレータアイテムの作成
--------------------------

`シミュレータアイテムの作成 <https://choreonoid.org/ja/manuals/latest/simulation/tank-tutorial/step1.html#id5>`_ を参照してください。

プロジェクトディレクトリの作成
------------------------------

`プロジェクトディレクトリの作成 <https://choreonoid.org/ja/manuals/latest/simulation/tank-tutorial/step2.html#id5>`_ を参照してください。

プロジェクトの保存
------------------

`プロジェクトの保存 <https://choreonoid.org/ja/manuals/latest/simulation/tank-tutorial/step1.html#id16>`_ を参照してください。

シミュレーションの実行
----------------------

`シミュレーションの実行 <https://choreonoid.org/ja/manuals/latest/simulation/tank-tutorial/step1.html#tank-tutorial-step1-start-simulation>`_ を参照してください。

シミュレーションの停止
----------------------

`シミュレーションの停止 <https://choreonoid.org/ja/manuals/latest/simulation/tank-tutorial/step1.html#id9>`_ を参照してください。
